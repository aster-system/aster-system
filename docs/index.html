Les produits Aster Système

PLEOS

PLEOS est un ensemble de logiciel, réalisé pour les professeurs / élèves et étudiants pour rendre l'apprentissage plus facile.
Plus d'informations sont disponibles juste ici

SAASF

SAASF est un site web regroupant pleins de cours sur divers sujets, allant de la physique quantique à l'informatique théorique.
Le site web de SAASF est disponible juste ici.

SCLS

SCLS est un ensemble de librairie C++, faites pour rendre le C++ aussi simple que Python.
La documentation de SCLS sera publiée prochainement.
