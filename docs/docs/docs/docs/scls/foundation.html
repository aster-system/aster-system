What is SCLS Foundation ?

SCLS Foundation is the core library of the Aster System SCLS project. It unites a lot of core functionnality to use each others SCLS libraries. It is name "Leonhard", after the Swiss mathematician Leonhard Euler, also known as one of the "Father of the modern mathematics". For now, the library is at its version 0.1.

What to do with SCLS Foundation ?

As mentionned in a next part, the license allows you to watch the code of the library. If you're are new with C++, you can learn from this library. As the creator of the library, I STRONGLY RECOMMEND you to watch the code and learn from it. It is made to be simple to read and user-friendly. If you know well C++, you can watch the code too, but it's certainly not the better code you have watched in your life. You can change and upgrade it if you want. If you publish it, we would be strongly grateful if you could mention us, or contact us to make your upgrade an official upgrade.

Documentation

The current documented version is the 0.1 version. Each SCLS Foundation functions / classes / ... uses the "scls" namespace, to be separate from the rest of your program and easily usable. SCLS Foundation is cutted in differents files, each ones having a specific usage. Here a list of each files :
  • scls_foundation.h : A file including every others files in the library.
  • scls_foundation_core.h : The core files and features of SCLS Foundation.
  • scls_foundation_binary.h : A simple and secure way to use binary with SCLS Foundation.
  • scls_foundation_file.html : A simple one-line way to handle easily files with SCLS Foundation.
  • scls_foundation_math.html : A sets of little simple mathematicals functions.
