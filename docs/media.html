Les médias Aster Système

En tant que seul moyen d'expression pour Aster Système, notre compagnie est très présente sur divers médias, et réseaux de partage d'informations. Si vous êtes sur cette page, il y a deux possibilités: vous venez d'un des médias de l'organisation, et vous voulez en savoir plus sur celle ci. Ou, vous avez entendu parler autre part de l'organisation, et voulez plus d'informations sur ses médias. Vous trouverez ici les ressources les plus susceptibles de vous intéresser, selon qui vous êtes.

Vous venez d'un média Aster Système

Les shorts / tiktoks / réels d'apprentissage sur Aster Système Learn

Si vous venez d'un média Aster Système Learn, il est fort probable que vous veniez d'un short / tiktok / réel d'apprentissage. Toutes les informations complémentaires se situent sur notre site web d'apprentissage : SAASF. Voici la classification des sujets que vous pourriez avoir vu passer :

  • La physique quantique : les atomes, le modèle standard, les particules élémentaires...
  • Les algorithmes : le pseudo-code, le tri, la complexité.
  • L'hardware : la carte mère, le processeur, la moire, le stockage...
  • L'impression 3D.
  • Le système d'exploitation : les OS, le Kernel, le stockage via le Kernel.

Les projets shorts / tiktoks / réels sur Aster Système

Si vous venez d'un média Aster Système, il y a de fortes chances que vous veniez pour en savoir plus sur un projet posté sous forme de short / tiktok / réel. Tous ces projets se situent sur notre site web d'apprentissage : SAASF. Voici la classification de toutes les pages des projets publié:

  • Le jeu "Snake" : présenté en détail sur SAASF.
